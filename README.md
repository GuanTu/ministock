# Quasar App (MiniStock)

A Quasar Framework app

## 安装依赖

```bash
yarn
```

### 运行

```bash
quasar dev -m electron
```

## 打包

```bash
quasar build -m electron
```

### 自定义

参考 [Configuring quasar.conf.js](https://quasar.dev).
